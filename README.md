# easyExamSystem轻松考系统

### 开发环境：
myeclipse+mysql+git

### 软件架构：
model+view+controller+easyui+bootstrap

### 项目描述：
本系统是基于B/S架构而设计开发，目的在于减少大学生考试挂科率，解决大学生考试不知道如何复习，找不到复习资料

### 系统主要模块：
- 资料下载上传：资料上传下载，用户积分相应变化等
- 提问与回答：用户提出问题，设置积分，别人回答或抢答
- 刷题：用户根据权限可以做相应的题目等
- 学习小组讨论：讨论回复学习方法等
